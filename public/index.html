Timestamp microservice

Endpoints:

https://sleepy-wildwood-95438.herokuapp.com/[timestamp]
where [timestamp] is a unix timestamp or a date string.

Example usage:

https://sleepy-wildwood-95438.herokuapp.com/December%2015,%202015
https://sleepy-wildwood-95438.herokuapp.com/1450137600

Example output:

{ "unix": 1450137600, "natural": "December 15, 2015" }
